⚙️ SUPABASE KURULUM REHBERİ
1
Supabase hesabı oluşturun
supabase.com'a gidin → "Start your project" → ücretsiz hesap açın → yeni proje oluşturun.
2
Veritabanı tablosunu oluşturun
Supabase paneli → SQL Editor → aşağıdaki kodu yapıştırın → Run:
CREATE TABLE push_subscribers ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, endpoint text UNIQUE NOT NULL, p256dh text NOT NULL, auth text NOT NULL, created_at timestamptz DEFAULT now() );
3
API anahtarlarınızı girin
Supabase → Settings → API → Project URL ve anon key'i kopyalayın. Aşağıya yapıştırın:
4
VAPID anahtarı oluşturun (Web Push için)
Terminalde çalıştırın (Node.js gerekli):
npx web-push generate-vapid-keys
Çıkan Public Key'i aşağıya girin:
5
Service Worker dosyasını oluşturun
Simülatörünüzü bir web sunucusunda yayınlıyorsanız, aşağıdaki sw.js dosyasını kök dizinine koyun:
self.addEventListener('push', e => { const d = e.data?.json() || {}; self.registration.showNotification( d.title || 'Yeni Halka Arz!', { body: d.body, icon: '/icon.png', badge: '/badge.png', tag: 'halkarz' } ); });
